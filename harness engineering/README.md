# Harness Engineering

This directory contains Chinese study notes for the paper:

**Agent Harness Engineering: A Survey**

Original sources:

- Project page: https://picrew.github.io/LLM-Harness/
- Paper PDF: https://picrew.github.io/LLM-Harness/main.pdf
- OpenReview: https://openreview.net/forum?id=eONq7FdiHa

## Files

- `agent_harness_engineering_survey_zh_detailed.pdf`: detailed Chinese study version, 35 pages.
- `agent_harness_engineering_survey_zh_detailed.md`: Markdown source for the detailed version.
- `agent_harness_engineering_survey_zh_detailed.html`: browser-readable detailed version.
- `agent_harness_engineering_survey_zh.pdf`: shorter Chinese study version.
- `agent_harness_engineering_survey_zh.md`: Markdown source for the shorter version.

These files are study notes and structured Chinese explanations, not a verbatim full-paper translation.
